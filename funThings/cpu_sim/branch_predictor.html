2-Bit Branch Predictor

Interaktywna wizualizacja automatu skończonego Smitha (2-bit saturating counter).

← Powrót do Symulatora CPU

Jak to działa?

Gdy procesor napotyka instrukcję skoku warunkowego (np. if-else w pętli), nie wie jeszcze, czy skok nastąpi, dopóki go nie policzy. Aby nie zatrzymywać potoku (Pipeline), Branch Predictor zgaduje. Najpopularniejszym algorytmem jest 2-bitowy licznik nasycający. Kliknij poniżej "Skok Wykonany" lub "Skok Niewykonany" aby zasymulować prawdziwy wynik instrukcji i zobacz jak predykator się uczy!
